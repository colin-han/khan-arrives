# 导出朋友圈配图截图

## 用法

运行打包好的脚本（脚本会自动去掉预览用的灰底与外边距，只截 1080×1440 画布本身）：

```bash
.claude/skills/export-image/scripts/export_image.sh <话题目录>
```

- `<话题目录>` 是形如 `2026-07-17-human-on-the-loop/` 的目录，也可以直接传 `image.html` 的路径。
- 输出固定为该目录下的 `image.png`。
- 默认 2 倍分辨率（2160×2880），手机上更清晰；如需 1:1 像素，加第二个参数 `1`。

## 流程

1. 确定当前话题目录：优先用本次会话正在制作的目录；若不明确，取仓库中最新的 `{YYYY-MM-DD}-{slug}/` 目录，并向用户确认。
2. 运行脚本，脚本会输出生成路径和实际像素尺寸。
3. 用 Read 查看生成的 `image.png` 自查一遍（是否截全、无灰边），然后 `open` 该 PNG 给用户确认。

## 注意

- 脚本假设 image.html 遵循本仓库设计规范：`.canvas` 为 1080×1440 固定画布。若画布类名或尺寸有变，需要相应调整脚本参数或临时改脚本。
- 依赖本机已安装的 Google Chrome，无需安装其它工具。
